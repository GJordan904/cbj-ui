CBJ UI

Angular 5 modules for common UI elements and Bootstrap 4 components, all with a beautiful Material Design look. Everything is pure Angular, no jQuery and no 3rd party dependencies.

This project aims to provide a few things:

  • Bootstraps jQuery Functionality
  • Additional Common UI Components & Directives
  • Material Design Theme

Angular

Built from the ground up using the latest Angular CLI, CBJ UI has no external JS dependencies but instead leverages the power of native Angular functionality.

Bootstrap 4

I utilize Bootstrap 4 for the grid and base styles with some minor adjustments to get the material design look I'm going for.

Material Design

The Material Design spec heavily influenced my styles for this project and I try to stick to the spec throughout.
