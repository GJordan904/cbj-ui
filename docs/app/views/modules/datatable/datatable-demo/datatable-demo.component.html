Data Table Component A Simple Data Table

Overview

Outside of using Angular, I would frequently use the jQuery Data Tables plugin and found it very useful. For this project, I didn't want to have any external dependencies and I certainly wasn't going to use jQuery, so rolling my own was the answer. Currently it is a very simplistic data table but I plan to develop it more and most likely move it to its own package.

The component has a single attribute, options, that is used to pass the data and configuration options to the table. This attribute should utilize the DatatableOptions interface.

Data Source

The table accepts either an array of data, an array of Objects using the DatatableMultiData interface, or a url string to request data from. Regardless of the data source, the structure of the data itself is not implied and to properly present the data, a coordinating columns array must exist for each data source.

Columns
