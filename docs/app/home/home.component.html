Angular, Bootstrap, and Material Design. Oh My!

Get Started Github

Powered by Angular

With no external dependencies aside from Angular, CbJ Ui is built entirely using Angular and uses Angular best practices. The distributable has been built according to the Angular Package Specifications and as such, is provided in the formats commonly used by today's web apps.

Styled with Bootstrap

Styled using the newest Bootstrap with all the flexbox niceties, CbJ Ui provides a convenient and familiar set of styles. Bootstrap 4 is actually built into the package and the SCSS as well as a compiled stylesheet are provided.

Inspired by Material Design

The look of the package is heavily inspired by the Material Design spec and uses and includes the material icons font family.
