CbJ Button

The <cbj-button></cbj-button> provides a convenient way of adding various styles of buttons to a page. Buttons are either <a> or <button> tags, depending on the options passed. Buttons will be in all caps by default as set by the sass variable $btn-text-transform.

Buttons are one of the components where I have completely overwritten the Bootstrap styles. I utilize a variation of the button-variant mixin and do not import the bootstrap button scss file. So if you were to use the btn class without the angular component, you would still get the same styles.
